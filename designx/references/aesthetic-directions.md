# Aesthetic Directions

Use a direction that fits product, platform, audience, and density. Merge directions only when the product needs mixed qualities.

Hybrid format:

```txt
DIRECTION:
BASE:
BORROWED TRAITS:
DO NOT BORROW:
PROTECTED RULES:
WHY:
```

Vary composition across tasks by changing at least three: layout archetype, nav model, density, type, palette, component geometry, data presentation, interaction model, media treatment, or content rhythm.

## Catalog

Brutal Tool: stark contrast, sharp geometry, mono type, exposed structure. Use for hacker tools, devtools, experimental interfaces.

Editorial Serif: strong serif display, generous rhythm, image-led hierarchy. Use for publications, portfolios, luxury narratives, research pages.

Soft Consumer: warm palette, rounded controls, friendly spacing. Use for wellness, family apps, lifestyle, personal finance.

Data Terminal: dark/neutral, mono accents, dense grids, status colors. Use for monitoring, infra, logs, trading.

Luxury Refined: restrained palette, premium type, large imagery, precise spacing. Use for luxury commerce, architecture, premium finance.

Retro Tech: pixel/CRT/synth references, chunky controls, playful motion. Use for audio tools, retro games, creative instruments.

Glass Depth: translucent layers, blur, luminous accents. Use for media, consumer dashboards, fintech/crypto only when brand supports it. Avoid dense enterprise.

Swiss Grid: strict grid, strong alignment, restrained palette, confident type. Use for design-led SaaS, serious product tools, portfolios.

Organic Natural: earth tones, soft geometry, tactile media. Use for sustainability, agriculture, wellness, food.

Neon Cyberpunk: dark base, electric accents, high contrast. Use for gaming, esports, security, nightlife, speculative work.

Minimal Functional: quiet UI, high utility, low ornament. Use for productivity, writing tools, internal systems, devtools.

Industrial Operations: compact, rugged, clear statuses, map/table/schedule emphasis. Use for construction, logistics, manufacturing, field management.

Calm Enterprise: restrained color, predictable layout, high legibility. Use for B2B SaaS, HR, procurement, finance ops.

AI Workspace: split-pane conversation/artifact/tool trace, trust indicators, source panels. Use for agents, assistants, automation tools.
